# Smart Healthcare Prediction System — Diabetes Risk Assessment

A machine learning web application that predicts diabetes risk (Non-Diabetic / Pre-Diabetic / Diabetic) from patient clinical data. Built with a 4-model ensemble pipeline and a Flask web interface.

## Model Performance

Trained on the UCI Diabetes 130-US Hospitals dataset (1999–2008). Class imbalance handled with SMOTE. Features normalized with StandardScaler.

| Model | Test F1 | CV F1 (5-fold) |
|---|---|---|
| Random Forest | 0.5677 | 0.7073 |
| Gradient Boosting | 0.5992 | 0.7225 |
| AdaBoost | 0.4765 | 0.5701 |
| **Voting Classifier** | **0.6016** | **0.7196** |

The Voting Classifier (soft voting across RF + GB + AdaBoost) achieves the highest test F1 and was selected for deployment.

## Quick Start

### Prerequisites
- Python 3.8+
- pip

### 1. Clone the repository
```bash
git clone https://github.com/muhammadhassan-ai/machine-learning-project.git
cd machine-learning-project
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train models and generate plots
```bash
python pipeline.py
```
This will:
- Load and preprocess `data/diabetes_raw.csv`
- Apply SMOTE for class rebalancing
- Train all 4 models
- Save serialised artifacts to `models/`
- Generate all EDA and evaluation plots to `plots/`

### 4. Start the web app
```bash
cd webapp
python app.py
```
Open `http://localhost:5000` in your browser.

## How It Works

1. **Data preprocessing** — categorical encoding, StandardScaler normalisation, SMOTE oversampling on the minority class.
2. **Model training** — Random Forest, Gradient Boosting, and AdaBoost trained independently; combined into a soft-voting Voting Classifier.
3. **Inference** — user submits patient features via the web form; Flask backend applies the saved scaler + encoders, runs the Voting Classifier, and returns a risk class with confidence score.

**Output classes:**
- `Non-Diabetic` — No diabetes markers detected.
- `Pre-Diabetic` — Pre-diabetic indicators present; lifestyle changes recommended.
- `Diabetic` — Diabetic indicators detected; medical consultation advised.
